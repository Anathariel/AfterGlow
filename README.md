# afterglow
Solo project on the theme of Cocktails; using the cocktailDb API.
Technology used : Vue.js; Vue Router; Node.js; SCSS / HTML / JS

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```
